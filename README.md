# Ejercicio lanzamiento de dados.

El actual repositorio trae con sigo un ejercicio en el que se realiza una simulación de un lanzamiento de dados donde indica el ganador respectivamente, esto lo realiza con la particularidad de hacer uso de la programación funcional, lo que trae con sigo la imposibilidad de usar estados y de esta forma evitar los efectos de lado.
